= CLI Headless Mode
:toc:

== Define Purpose

Document headless generation execution paths and diagnostics.

== Define Inputs

* CLI options (`generate`, `--dry-run`, metadata overrides, extensions)
* Runtime config and timeouts

== Define Outputs

* Exit codes
* Generated project files (non-dry-run)
* Diagnostic output (standard and verbose)

== Define Failure Modes

* Metadata/catalog timeout
* Validation failure
* Archive or download failure
