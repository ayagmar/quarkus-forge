== Core Capabilities

* Guided TUI with fast extension search, fuzzy highlighting, and chip-style selection badges.
* Extension descriptions shown inline from the Quarkus API.
* Headless command for CI and scripts.
* Unified Forgefile format for shareable, deterministic project templates.
* Safe archive extraction and deterministic fallback behavior for metadata/catalog loading.
* Configurable theming and IDE command for terminal readability.
* Session persistence (remembers your last selections).
