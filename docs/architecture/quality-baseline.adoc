== Duplication Hotspots
1. Key matcher duplication in `CoreTuiController` and `ExtensionCatalogState` (`isPlainChar`, vim movement checks).
2. Throwable unwrap duplication (`CompletionException`/`ExecutionException`) in `QuarkusForgeCli`, `CoreTuiController`, `CatalogDataService`, `QuarkusApiClient`.
3. App-data root and atomic write duplication (`~/.quarkus-forge` + `ATOMIC_MOVE`) in `CatalogSnapshotCache`, `UserPreferencesStore`, and `ExtensionFavoritesStore`.
